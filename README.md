# Dapoer Resto
    Aplikasi manajemen restoran menggunakan PHP dan MySQL.

## Fitur
    - Manajemen Menu Makanan dan Minuman
    - Manajemen Reservasi Meja
    - Manajemen Order
    - Manajemen Dapur
    - Manajemen User
    - Report Transaksi
